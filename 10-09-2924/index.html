Belajar HTML dan CSS

HTML untuk membuat tampilan website, CSS untuk mengatur tampilan website

Materi HTML

  • h1 untuk judul
  • p untuk paragraf
  • img untuk gambar
  • table untuk tabel
  • Input untuk form
  • li untuk list atau daftar

Materi CSS

  1. Margin,Padding,Position untuk mengatur jarak antara elemen HTML
  2. Display untuk mengatur elemen HTM agar tampil atau tidak tampil, mendatar atau menurun
  3. Style untuk mengatur form
